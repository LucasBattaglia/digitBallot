# Boîte à Jeu

Bienvenue dans la Boîte à Jeu ! Un site web contenant une base de jeux pour vous permettre de preparer vos animations.

## Description

La Boîte à Jeu est un site web développé en HTML, CSS et JavaScript. Il offre une collection de jeux divertissants accessibles directement depuis votre navigateur.

## Jeux Disponibles

1. La Balle Au Prisoners
2. Catch Flag
3. Sagamore

## Comment exécuter

1. Télécharger le fichier sur gitlab ou en .zip
2. Depuis le terminal, rendez-vous sur le repertoire Web du projet
3. Lancer le serveur avec la commande ```./httpd``` et récupérer Le port du serveur
4. Dans votre navigateur, chercher : ```localhost:XXXX``` où XXXX est le numéro de port
